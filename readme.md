## Install

1. [install](https://gohugo.io/installation/) hugo
1. clone repo
1. run `npm i`
1. run `hugo server`

## How to

1. Edit _content_index.md_ to switch on/off **tailwind** or **bootstrap**

   ```
   tailwind: true
   alpine: true
   bootstrap: true
   jquery: true
   ```

1. Add you html code in **layouts\index.html**
1. Check the style **generated**
